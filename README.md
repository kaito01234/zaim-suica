# zaim-suica

Suica（交通系IC）の利用明細 CSV を **Zaim** に取り込むツール。

```
日付,内容,金額
2021/11/17,入 秋葉原 (つくばエクスプレス)  出 八潮 (つくばエクスプレス),-471
2021/11/17,チャージ 携帯電話,648
2021/11/13,物販 物販端末,-750
```

このフォーマットの CSV を読み取り、内容から取引種別を判定して Zaim に登録します。

| CSV の内容 | 判定 | Zaim への登録 |
| --- | --- | --- |
| `入 …出 …` | 乗車 | **支出**（交通 / 電車、口座=Suica） |
| `物販 …` | 買い物 | **支出**（食費 / その他、口座=Suica） |
| `チャージ …` | チャージ | **収入**（既定）または**振替**。`ZAIM_CHARGE_MODE` で切替 |

## 構成

Google Drive の特定フォルダに置かれた CSV を定期ポーリングで検知し、
**新規取引だけ**を抽出して Zaim に自動登録する **Google Apps Script** 版です。
過去分と重複する CSV でも取込ログ（Spreadsheet）と突き合わせて差分だけを登録します。

→ セットアップ・使い方・仕組みは **[gas/README.md](gas/README.md)** を参照してください。

- [gas/](gas/) … GAS プロジェクト本体
- [2026-07-13.csv](2026-07-13.csv) … 動作確認用のサンプル CSV
- `.env` … Zaim の認証トークン・マッピング設定の控え（GAS のスクリプトプロパティに転記して使う。`.gitignore` 済み）

## 注意

- `.env` / `id.txt` は `.gitignore` 済み。**認証情報は絶対にコミットしないこと。**
- Zaim API の日付は「過去・未来 5 年以内」が有効です（それ以前の明細は登録に失敗します）。
